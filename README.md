# Alpha-APK
Développer une application capable de générer automatiquement des épreuves similaires à partir de sujets existants afin d’aider les étudiants à mieux se préparer aux examens. L’application analysera la structure, le niveau et les thèmes des épreuves pour créer de nouvelles versions avec corrigés, permettant un entraînement personnalisé et illimité.
